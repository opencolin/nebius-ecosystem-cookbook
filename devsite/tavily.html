Web search API

Give your agents real-time web access with Tavily.

Tavily is the search and retrieval API built for LLMs and agents. Search, extract, crawl, and map the web with one call, then ground your Nebius-powered models on fresh, reliable sources. Free to start — no credit card required.

One API, four primitives

Everything an agent needs to read the web.

Search

One call returns ranked, LLM-ready results with sources and an optional AI answer.

Extract

Pull clean, parsed content from any URL — no scraping boilerplate.

Crawl

Traverse a site from a starting URL and collect pages in structured form.

Map

Discover a site's link graph to plan retrieval and coverage.
